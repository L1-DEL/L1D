⚙️ 配置 GitHub Token

发布功能需要下面这个权限来把文章推送到GitHub。Token只授权给这一个仓库，不会碰到你其他项目：

  1. 打开 github.com/settings/tokens
  2. 点击 Generate new token → Fine-grained token
  3. Repository access: Only select repositories → 选 L1-DEL/L1D
  4. Permissions: Contents → Read and write
  5. 点 Generate，复制生成的Token
Token无效或没有权限
